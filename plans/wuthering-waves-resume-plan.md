# План разработки Сайта-Резюме в стиле Wuthering Waves

## 1. Обзор проекта (Objective)
Создать интерактивный, некоммерческий сайт-резюме в стиле игрового меню (вдохновлено игрой Wuthering Waves). Сайт должен эффектно презентовать навыки и опыт разработчика (более 10 лет опыта, фокус на BMC Helix, БД, Docker/Kubernetes, Angular, Java), используя геймифицированный подход к подаче информации (экраны загрузки, карточка персонажа, инвентарь проектов).

## 2. Технологический стек
*   **Фреймворк:** Angular (последняя стабильная версия)
*   **Язык:** TypeScript
*   **Стилизация:** SCSS + TailwindCSS (для быстрой верстки и UI-компонентов)
*   **Анимации:** GSAP (для сложных переходов и "игровых" эффектов) и/или Angular Animations
*   **Ассеты:** Изображения и видео из открытых источников (с эффектами параллакса и CSS-фильтрами)

## 3. Архитектура и Структура (Scope & Impact)

Сайт будет представлять собой SPA (Single Page Application) с роутингом между "игровыми экранами".

### Экраны (Страницы):
1.  **Splash Screen (Экран загрузки):**
    *   Фоновое видео или красивый арт.
    *   Текст "Click to Start" или "Press Any Key".
    *   Анимация перехода в главное меню (вспышка, эффект глитча).
2.  **Main Menu (Главное меню / Terminal):**
    *   Боковая или верхняя панель навигации в футуристичном стиле.
    *   Переключение между вкладками без перезагрузки (плавный fade/slide).
3.  **Вкладки меню:**
    *   **Profile (Обо мне):** "Карточка персонажа". Имя, класс (Fullstack/Helix Engineer), уровень (10+ лет опыта). Аватар в стиле игрового портрета.
    *   **Attributes/Skills (Навыки):** Визуализация стека (BMC Helix, PostgreSQL, MSSQL, Oracle, Docker, Kubernetes, Java, Angular) в виде диаграмм прокачки (радары, прогресс-бары).
    *   **Inventory/Echoes (Опыт и Проекты):** Опыт работы и плагины оформлены как предметы в инвентаре. При клике — модальное окно с деталями "артефакта".
    *   **Map/Comms (Контакты):** Ссылки на GitHub, Telegram и т.д. в виде элементов карты или терминала связи.

## 4. Этапы реализации (Implementation Plan)

### Этап 1: Инициализация и настройка (Setup)
*   [ ] Создание нового проекта Angular (`ng new my-site`).
*   [ ] Установка и настройка TailwindCSS.
*   [ ] Установка GSAP.
*   [ ] Настройка базовой структуры папок (компоненты, сервисы, ассеты).

### Этап 2: Базовые компоненты и Layout
*   [ ] Создание компонента Layout (обертка для игрового интерфейса).
*   [ ] Создание компонента навигации (меню).
*   [ ] Настройка Angular Router для перехода между "вкладками" меню.

### Этап 3: Реализация экранов
*   [ ] **Splash Screen:** Разработка стартового экрана с анимацией по клику.
*   [ ] **Profile View:** Верстка карточки персонажа с использованием CSS Grid/Flexbox и Tailwind.
*   [ ] **Skills View:** Создание компонентов для отображения навыков (прогресс-бары или радар-чарты, возможно использование Chart.js или кастомного CSS).
*   [ ] **Projects/Inventory View:** Создание сетки "предметов" и компонента модального окна для деталей проекта.
*   [ ] **Contacts View:** Верстка экрана контактов.

### Этап 4: Анимации и Полировка (Polish)
*   [ ] Интеграция GSAP для анимаций переходов между страницами.
*   [ ] Добавление эффектов наведения (hover effects), звуковых эффектов (опционально, при наведении/клике).
*   [ ] Поиск и интеграция красивых фоновых изображений/видео (ассеты из интернета).
*   [ ] Оптимизация под мобильные устройства (responsive design), хотя основной фокус на десктоп.

## 5. Проверка (Verification)
*   Проверка работы всех роутов.
*   Проверка плавности анимаций и отсутствия лагов.
*   Тестирование адаптивности на разных разрешениях экрана.

## 6. Дальнейшие шаги (После завершения плана)
*   Развертывание проекта (напр. GitHub Pages, Vercel).
